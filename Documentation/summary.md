### Given by the title of the repo, this final project will attempt to try and replicate the cool and still very hip game of "Battleship" in Python!

While making this code, I will be implementing things that I have learned over the course of my education at UNK such as OOP principles, data structures,
string parsing, and more! The user will play against a machine that will use algorithms (of which are still being decided) to try and compete with the player to win the game. The player will fight against the bot to sink each other's ships using a graphic interface to display it.
